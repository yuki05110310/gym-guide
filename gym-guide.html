info_outline 姿勢改善 + 代謝UP。見た目が大きく変わる部位
ラットプルダウン fitness_center マシン
repeat 3セット x 15回
help_outlineやり方を見るexpand_more
  1. シートに座り、太ももをパッドで固定する
  2. バーを肩幅より少し広めで握る
  3. 胸を張って、少し後ろに体を倒す
  4. バーを鎖骨のあたりに向かって引き下ろす
  5. 肩甲骨を「寄せる」意識で引く
  6. ゆっくり3秒かけて戻す
tips_and_updates
ポイント: 腕で引くのではなく「背中で引く」意識。肘を後ろに引くイメージ。
シーテッドロウ fitness_center マシン
repeat 3セット x 15回
help_outlineやり方を見るexpand_more
  1. シートに座り、胸をパッドにつける
  2. ハンドルを両手で握る
  3. 胸を張ったまま、ハンドルを体に引き寄せる
  4. 引いた時に肩甲骨をギュッと寄せる
  5. 肘が体の横を通るように引く
  6. ゆっくり戻す
tips_and_updates
ポイント: 背中の真ん中を「ぎゅっ」と締める感覚。肩が上がらないように注意。
チンアシスト(懸垂補助) fitness_center マシン
repeat 3セット x 8〜10回
help_outlineやり方を見るexpand_more
  1. 重りを選ぶ(重いほど補助が強い=楽になる)
  2. 上のチンアップバー(広いバー)を肩幅より少し広めで握る
  3. 膝パッドに両膝を乗せる
  4. 胸を張って、体を引き上げる
  5. あごがバーの高さまで来たら1秒キープ
  6. ゆっくり3秒かけて下ろす
tips_and_updates
ポイント: 背中全体に効く最強種目。最初は補助重量を重めにして、徐々に軽くしていく。
バックエクステンション(体幹・腰) fitness_center マシン
repeat 3セット x 15回
help_outlineやり方を見るexpand_more
  1. パッドに太ももの付け根を当てて、足をプレートに固定
  2. 腕を胸の前でクロスする
  3. 上体をゆっくり前に倒す(45〜60度くらい)
  4. 背中の筋肉を使って元の位置まで戻す
  5. 体が一直線になったら止める(反りすぎない)
  6. ゆっくり繰り返す
warning
注意: 反りすぎると腰を痛める。体が一直線になったらストップ。勢いをつけずゆっくり。

WEDNESDAY

水曜日 - 脚 + 体幹
info_outline 脂肪燃焼に最も効く日。体最大の筋肉を鍛える
レッグプレス fitness_center マシン
repeat 3セット x 15回
help_outlineやり方を見るexpand_more
  1. シートに深く座り、足をプレートに肩幅で置く
  2. つま先は少し外側に向ける
  3. 安全ハンドルを内側に回してロック解除
  4. ゆっくり3秒かけて膝を曲げ、脚を体に近づける
  5. 膝が90度くらいまで来たら、足裏全体で押し上げる
  6. 膝を完全に伸ばしきらない(少し曲がった状態で止める)
tips_and_updates
ポイント: かかとで押す意識。膝がつま先より内側に入らないように注意。

  4. できるだけ大きく開いたら、1秒キープ
  5. ゆっくり戻す(バタンと戻さない)
tips_and_updates
ポイント: お尻の横の筋肉を意識する。反動を使わずゆっくり。

土曜日 - 胸 + 肩 + 下腹
info_outline 上半身前面 + 下っ腹を集中的に攻める日
ペックフライ(胸) fitness_center マシン
repeat 3セット x 15回
help_outlineやり方を見るexpand_more
  1. シートに座り、背中をしっかりつける
  2. グリップを両手で握る(肘が肩と同じ高さ)
  3. 胸の前で両腕を閉じるように動かす
  4. 閉じきったら胸をギュッと絞る意識で1秒キープ
  5. ゆっくり戻す(胸のストレッチを感じる)
  6. 戻しすぎない。肩より後ろに行かない程度で
tips_and_updates
ポイント: 腕を閉じる時に「胸で挟む」イメージ。肩に力が入らないよう注意。
ディップアシスト(胸下部・三頭筋) fitness_center マシン
repeat 3セット x 10回
help_outlineやり方を見るexpand_more
  1. 重りを選ぶ(重いほど補助が強い=楽になる)
  2. 横のディップバー(平行バー)を握る
  3. 膝パッドに両膝を乗せる
  4. 体を少し前傾させる(胸に効かせるため)
  5. 肘を曲げてゆっくり体を下ろす(肘90度まで)
  6. 押し上げて元に戻す
tips_and_updates
ポイント: 前傾すると胸に、直立だと三頭筋に効く。胸の日なので前傾気味で。月曜のチンアシストと同じマシン。
リアデルト(肩の後ろ) fitness_center マシン
repeat 3セット x 15回
help_outlineやり方を見るexpand_more
  1. ペックフライの逆向きにセット
  2. 胸をパッドにつけ、グリップを握る
  3. 腕を後ろに大きく開く
  4. 肩甲骨を寄せるように、肩の後ろで引く意識
  5. 開ききったら1秒キープ
  6. ゆっくり戻す
tips_and_updates
ポイント: 軽い重量でOK。ペックフライと同じマシンの逆向き。肩の後ろの小さい筋肉なので無理しない。
レッグレイズ(下腹) self_improvement 床
repeat 3セット x 10回
self_improvement
help_outlineやり方を見るexpand_more
  1. 仰向けに寝る。両手はお尻の横か、お尻の下に敷く
  2. 両足を揃えてまっすぐ伸ばす
  3. 足を床からゆっくり持ち上げる(90度まで上げなくてOK、60度くらい)
  4. ゆっくり3秒かけて下ろす(ここが一番効く)
  5. 足が床につく直前で止める(床につけない)
  6. そのまま次の1回を持ち上げる
warning
注意: 腰が浮いたら腰痛になる。腰を床に押し付ける意識が超重要。きつかったら膝を曲げてもOK。
tips_and_updates
下っ腹に最も効く種目。 ゆっくり下ろすほど効果大。

ROADMAP

目標達成までのロードマップ

72kg → 62kg / 健康的なペースで約4〜5ヶ月

1

1ヶ月目: 体が慣れる期間

体重: 72kg → 70kg (-2kg)
筋肉痛が出るが徐々に慣れる。フォームを覚えることが最優先。食事の見直しを始める。この段階では体重が落ちなくても焦らない。

2

2ヶ月目: 体が変わり始める

体重: 70kg → 67kg (-3kg)
筋肉がついて基礎代謝が上がる。重量を少しずつ増やせる。ランニングも楽になってくる。服のフィット感が変わり始める。

3

3ヶ月目: 周囲が気づく変化

体重: 67kg → 65kg (-2kg)
下っ腹が目に見えて減る。体重の減少ペースはゆるやかになるが、体脂肪率は下がり続ける。見た目の変化が大きい時期。

4-5

4〜5ヶ月目: 目標達成

体重: 65kg → 62kg (-3kg)
トレーニングが習慣化。重量も初期の1.5〜2倍に。下っ腹はかなりスッキリ。ここからは維持 + さらなるボディメイクへ。

restaurant 食事のポイント
体重を落とすのは「運動3割 : 食事7割」
  • check_circle
    タンパク質を毎食摂る - 鶏むね肉、卵、豆腐、プロテイン
  • check_circle
    白米を少し減らす - いきなりゼロにしない。2/3くらいに
  • check_circle
    揚げ物・菓子パンを減らす - 週1〜2回のご褒美はOK
  • water_drop
    水を1日2L飲む - 代謝UPに直結
  • schedule
    夜21時以降は食べない - これだけでも効果あり
tips_and_updates
最重要: 極端な食事制限は絶対NG。筋肉が落ちてリバウンドする。「少し減らす」を継続する方が10倍効果的。
